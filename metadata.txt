about=This plugin allows you to download Italian cadastral parcel data from Zornade's enriched cadastral dataset via RapidAPI directly into QGIS. The plugin provides access to enriched cadastral parcels with accurate geometries, administrative information, elevation data, land cover classification, and comprehensive risk assessments including flood, landslide, and seismic risks. Features include: automatic coordinate transformation, secure credential storage, batch processing with concurrent downloads for improved performance, comprehensive error handling, and seamless integration with the QGIS Processing Framework. Simply provide your RapidAPI credentials and define a bounding box to fetch detailed parcel geometries and attributes for analysis and visualization.
